# AH. ビジュアルデザイン

## 概要

ビジュアルデザインは、色・フォント・レイアウト・余白を統制し、ブランド一貫性を保ちながらユーザーが直感的に理解できるUIを実現します。

見た目は単なる装飾ではなく、ユーザー行動をガイドする言語です。

---

## 1. カラーシステム

**階層構造:**
- **プライマリカラー** - ブランドの核（ロゴ、CTA）
- **セカンダリカラー** - 補完的（成功状態、確認）
- **ニュートラルカラー** - テキスト・背景・ボーダー
- **ステータスカラー** - 成功（緑）、警告（黄）、エラー（赤）、情報（青）

**実装:**
```css
:root {
  --color-primary: #0052CC;
  --color-success: #28A745;
  --color-danger: #DC3545;
  --color-text: #333333;
  --color-bg: #F5F5F5;
}
```

**確認項目:**
- コントラスト比が4.5:1以上か（WCAG AA基準）
- ダークモードに対応しているか

---

## 2. タイポグラフィ

**フォント選定:**
- **見出し用** - Segoe UI, Verdana等（ブランド性重視）
- **本文用** - -apple-system, Roboto等（読みやすさ重視）
- **コード用** - Menlo, Monaco等（等幅）

**サイズ体系（モバイルファースト）:**
```
h1: 28px (モバイル) / 36px (デスクトップ)
h2: 24px / 28px
h3: 20px / 24px
body: 14px / 16px
caption: 12px / 13px
```

**確認項目:**
- 本文は最低12px以上（モバイル）か
- 見出しとの大きさ差は1.5倍以上か
- 行高は本文で1.5以上か
- 使用フォントは3種類以下か

---

## 3. レイアウト

**グリッドシステム:**
- **モバイル**（0～480px）：4カラム
- **タブレット**（768px～）：8カラム
- **デスクトップ**（1024px～）：12カラム
- **大画面**（1440px～）：12カラム + max-width制限

**グリッド間隔:**
```css
/* モバイル */
.container {
  grid-template-columns: repeat(4, 1fr);
  gap: 16px;
  padding: 0 16px;
}

/* デスクトップ */
@media (min-width: 1024px) {
  .container {
    grid-template-columns: repeat(12, 1fr);
    gap: 24px;
    padding: 0 32px;
  }
}
```

---

## 4. スペーシング（余白）

**8pxベーススケール:**
```
xs: 4px     (細かい調整)
sm: 8px     (関連要素同士)
md: 16px    (要素間)
lg: 24px    (セクション間)
xl: 32px    (大セクション)
```

**視覚的な関連性:**
- ラベルと入力欄：小さな余白（8px）
- フォーム項目同士：大きな余白（24px）

---

## 5. レスポンシブデザイン

**モバイルファースト:**
1. モバイル（320px～）：1カラム、要素を縦に積み上げ
2. タブレット（768px～）：2カラム、サイドバー追加
3. デスクトップ（1024px～）：複数カラムグリッド

**ダークモード対応:**
```css
@media (prefers-color-scheme: dark) {
  body {
    background-color: #1a1a1a;
    color: #e0e0e0;
  }
}
```

- [ ] コントラスト比が4.5:1以上か
- [ ] フォントは3種類以下、ウェイト4種類以下か
- [ ] グリッドシステムに沿っているか
- [ ] スペーシングが8pxベースか
- [ ] すべてのデバイスサイズでテストしたか
