## 学習のアドバイス

### Sprint 1での心構え

```
┌─────────────────────────────────────────────────────┐
│  補足項目の扱い方                                    │
│                                                     │
│  ✅ 「存在を知る」レベルでOK                         │
│     → 「こういう概念があるんだな」程度               │
│                                                     │
│  ✅ 実装中に出てきたら調べる                         │
│     → AIに聞きながら進める                          │
│                                                     │
│  ✅ 深掘りは後のSprintで                            │
│     → 今は重点項目に集中                            │
│                                                     │
│  ❌ 全部理解しようとしない                          │
│     → 完璧主義は学習の敵                            │
└─────────────────────────────────────────────────────┘
```

### 「分からない」への対処法

1. **AIに聞く**：「〇〇とは何ですか？初学者向けに説明してください」
2. **動かしてみる**：理論より実践。動かすと分かることが多い
3. **後回しにする**：今分からなくても、後で分かることも多い
