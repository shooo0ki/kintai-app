# E. プログラミング基礎

## 概要

変数・型・制御構文・関数は、すべての開発の土台です。これらを理解することで、AI生成コードを正しく検証・修正できます。

---

## 1. 変数と型

変数はデータを保存する「箱」です。型システムがあると、バグを早期に発見できます。

**主な型：** 数値、文字列、ブーリアン、配列、オブジェクト、null/undefined

スコープと const/let/var の違いを理解することが重要です。const は再代入不可、let はブロックスコープです。

---

## 2. 制御構文

**if/else：** 条件に応じて処理を分岐。複雑な条件は中間変数で意図を明確にします。

**ループ：** for（回数が決まっている）、while（条件で判定）、for...of（要素を順番に）

**switch：** 複数の値で分岐する場合に使用

```javascript
const isAdult = user.age > 18;
const isActive = user.status === 'active';
if (isAdult && isActive) {
  allowAccess();
}
```

---

## 3. 関数

関数は「入力→処理→出力」の単位です。引数と戻り値で入出力を明確にし、デフォルト引数でオプション処理に対応します。

---

## 4. スコープ

グローバル（全体）、ローカル（関数内）、ブロック（{}内）の3つがあります。変数を必要な範囲だけに限定することで、バグを防止します。

---

## 5. 例外処理

try/catch で予期しないエラーに対応します。finally 句で必ず実行すべき処理を記述します。

```javascript
try {
  riskyOperation();
} catch (error) {
  console.log('Error:', error.message);
} finally {
  cleanup();
}
```

---

## 6. 配列とオブジェクト

**配列：** 複数の値を順序付けて格納。map/filter/reduce で効率的に変換・選別・集約できます。

**オブジェクト：** キーと値のペアでデータを組織。構造化されたデータ管理に適します。

---

## まとめ

**重点項目：** 変数・型、制御構文、関数、スコープ、例外処理、データ構造。これらが確実に身につくことで、高度なパラダイムへの学習効率が高まります。
